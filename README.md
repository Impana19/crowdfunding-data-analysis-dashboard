# crowdfunding-data-analysis
Crowdfunding Data Analysis Dashboard
Project Overview

This project analyzes crowdfunding campaign performance using data visualization techniques to identify funding trends, success rates, and category performance.

Objectives

Analyze campaign success patterns

Identify funding trends

Visualize key performance metrics

Tools Used

Excel

Power BI

Tableau

Data Analysis

Key Insights

Identified high-performing categories

Analyzed funding success factors

Created interactive dashboards
